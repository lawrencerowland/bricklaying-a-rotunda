This is a visual analogy for project planning under partial visibility:

  • The circular path is the full project state space.
  • The viewing arc is the limited information you get at decision points.
  • Changing speed or start point is like adjusting resources or sequencing.

It shows how hidden parts of the process can affect outcomes, and why mapping input–output relationships under constraints is critical for steering complex projects.

We built a visual “lens” model of an open dynamical system — an object moving in a circle, observed only through a small visible arc — to map how inputs (initial speed, position, radius) produce outputs (observed positions and timings).

This formalism uses category-theoretic lenses to link the system’s state space (manifold) to its input/output behaviour, with wiring diagrams showing composable pathways.

The demo makes abstract maths tangible, giving PMs a clear, reusable blueprint for designing and simulating constrained process flows in any project environment.
